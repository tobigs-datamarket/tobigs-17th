# Who we are

## 안녕하세요, 대학생 빅데이터 연합동아리 **Tobigs** 입니다.

이 Repository와 Google Sites은 그 활동의 일부를 정리한 문서입니다.  

저희 투빅스는 매년 **6월, 12월** 마다 **신입 기수를 모집하고 있습니다.**
